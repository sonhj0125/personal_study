## 11. 기본형의 종류와 범위

* 기본형 = 8개의 타입(자료형) => 논리형, 문자형, 정수형, 실수형

* 정수형은 int가 기본, 실수형에서는 double이 기본

|종류/크기|1 byte|2 byte|4 byte|8 byte|
|:---:|:---:|:---:|:---:|:---:|
| 논리형 | boolean | | | |
| 문자형 | | char | | |
| 정수형 | byte | short | <span style="color:yellow">int</span> | long |
| 실수형 | | | float | *double* |
